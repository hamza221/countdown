## 2026-04-20 - Display Board Accessibility
**Learning:** In display-board style applications (like flight countdowns), screen readers get extremely noisy with rapidly ticking numbers and decorative animations. SVGs and Canvases representing visual fluff (airplanes, confetti) clutter the accessible tree. Crucial async status updates (delays, arrivals) often get missed if they just appear on screen without polite or assertive alerts.
**Action:** Always add `aria-hidden="true"` to decorative `<svg>` and `<canvas>` elements. Use `role="timer" aria-live="off"` for rapidly ticking countdowns to prevent overwhelming screen reader announcements, while allowing users to poll it manually. Use `role="alert" aria-live="assertive"` for critical asynchronous status updates like flight delays and final arrival messages.
