## 2023-10-27 - Flight Dashboard Accessibility
**Learning:** Highly visual real-time elements like flight trackers and countdowns are completely inaccessible to screen readers without proper ARIA states and roles. Duplicate IDs cause screen readers to fail completely when parsing elements, and pure visual indicators (like the route line and confetti) need to be explicitly hidden to avoid confusing screen reader output.
**Action:** When working on real-time tracking dashboards, always add `role="timer"` to countdowns, `role="status"`/`aria-live="polite"` to updates and loading screens, `role="alert"` for important changes (like delays), explicitly mark decorative animations as `aria-hidden="true"`, and ensure tab titles update dynamically to provide context for visually impaired users with multiple tabs open.
